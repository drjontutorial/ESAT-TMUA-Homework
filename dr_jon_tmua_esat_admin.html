💾 Backup

Downloads everything — all homework sets, student accounts, and every submission (including answers and the activity timeline) — as one JSON file to your computer. Since this data isn't protected by database rules, keeping your own copy is good practice.

⬆️ Restore from Backup

⚠️ Restoring overwrites whatever is currently in Firebase for any section present in the file (sets, students, and/or submissions). This can't be undone — only do this if you're sure, ideally right after taking a fresh backup first.

➕ Create Student Profile

Creates a new login for a student — they'll use this code + password to log in and see their approved papers.

📌 Approve Papers — Bulk

Pick a paper, tick students, and give them access in one go. This adds to what they already have — it won't remove any paper they can already see.
